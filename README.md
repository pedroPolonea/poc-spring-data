# poc-spring-data
Poc com o intuito de explorar as interações com o Spring Data e outras tecnologia.

# Requirements
* Spring Boot 2.x
* Java 11
* Lombok

# :baby: Baby steps 
* jUnit 5
* H2
* Spring Data
    * Query
    * Query Native
    * Example
        * ExampleMatcher
    * Specification
* Mapstruct
* Rest Assured
* Swagger
